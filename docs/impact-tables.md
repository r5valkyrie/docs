

PrecacheImpactEffectTable( string impactEffectTableName )

these contain:

decals / sprites for bullet impacts

impact particle effects 

audio event dictionaries for impact sounds depending on different surface types (.qc surfaceprop? vmt surfaceprop?)



Types:

- Bullet whizzby and impact tables
- Melee impact tables
- Pilot tables (footsteps, jumpkit boost jet washes, landings, slides, etc.)
- Titan tables (footsteps, weapons, cores, etc.)
- NPC tables (abilities, landings, etc. )
- Grapple tables
- Explosion tables
- Bounce tables
- Ragdoll tables 
- Ship tables (Dropship in Apex)
- Ability / Kit impact tables (i.e.: Pulse Blade)
